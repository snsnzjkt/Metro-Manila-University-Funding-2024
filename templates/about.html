About This Project

Project Overview

This project creates an interactive map visualization of research funding distribution across universities in Metro Manila. The visualization displays both public and private universities, with circle size representing funding amounts and colors differentiating between public and private institutions.

The goal of this project is to provide a clear, visual representation of how research funding is distributed across major educational institutions in Metro Manila, helping to identify patterns and potential areas for future investment.

Data Sources

The data for this visualization has been compiled from multiple sources:

  • Department of Science and Technology (DOST) research grant reports
  • Commission on Higher Education (CHED) institutional funding data
  • University annual reports and financial statements
  • Research publication databases for output metrics

Note: The current data represents estimated research funding based on publicly available information. For the most accurate figures, official data should be obtained directly from relevant institutions.

Technologies Used

This project was built using a stack of modern web technologies:

  • Python for data processing and backend logic
  • Flask web framework for serving the application
  • Folium for generating interactive maps
  • Pandas for data manipulation and analysis
  • HTML, CSS, and JavaScript for frontend functionality
  • Bootstrap for responsive design

Methodology

The map visualization process involved these key steps:

  1. Data Collection: Gathering funding information from various sources
  2. Data Cleaning: Normalizing and standardizing the collected data
  3. Geocoding: Adding precise geographic coordinates for each university
  4. Visualization Design: Creating an intuitive visual representation of the data
  5. Interactive Features: Adding tools for users to explore the data in different ways
